## Local version information

| Parameter      | Value                               |
| -------------- | ----------------------------------- |
| HCI Version    | 0x0d (5.4)                          |
| HCI Subversion | 0x0202 (514)                        |
| LMP Version    | 0x0d (5.4)                          |
| LMP Subversion | 0x0002 (2)                          |
| Company        | 0x08e7 (Barrot Technology Co.,Ltd.) |
